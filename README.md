# `txtinv.py`: Gradient-based Feature Inference Attack (Reconstruction Attack) for text classification tasks.

## How to run it?
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python ./txtint.py --help
```
